Security Notes

This is a tool for use in competitions, not a password manager.

Room link = room access

Anyone with the room UUID can open the board and decrypt its passwords, so treat the link itself like a secret.

Generator is convenience-focused

Built-in password generation is meant to be fast and useful during competitions, but it is not positioned as a replacement for a dedicated cryptographic password manager.

Boards expire automatically

Room data is deleted after 7 days, which helps reduce stale credential exposure after an event ends.
